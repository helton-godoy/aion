## üöÄ Getting Started

### Prerequisites
- Node.js >= 20.0.0
- Git
- GitHub account (for native integration)

### Installation
```bash
# Clone AION
git clone https://github.com/helton-godoy/aion.git
cd aion

# Setup BMAD foundation
npm run setup:bmad

# Setup GitHub integration
npm run aion:setup:github

# Initialize AION
npm run aion:init
```

### First Workflow
```bash
# Run autonomous development cycle
npm run aion:workflow:github-full-cycle

# Check memory status
npm run aion:memory:status

# List available agents
npm run aion:agents:list
```
